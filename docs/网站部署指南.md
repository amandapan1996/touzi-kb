## 二、第一步：本地测试（5分钟）

### 2.1 启动本地服务器

打开终端：

```bash
cd ~/Desktop/投资大佬agent知识库/website
python3 -m http.server 3001
```

### 2.2 打开浏览器测试

访问：http://localhost:3001/

检查页面是否正常显示。

---

## 三、第二步：获取 Cookie（5分钟）

### 3.1 打开浏览器

用 Chrome 打开 https://xueqiu.com 并登录

### 3.2 获取 Cookie

1. 按 `Command + Option + I` 打开开发者工具
2. 点击顶部 **Application** 标签
3. 左侧找到 **Cookies** → 点击 `https://xueqiu.com`
4. 找到 `xq_a_token` 这一行
5. 复制 **Value** 列的值（一长串）

### 3.3 把 Cookie 发给我

复制后发给我，我帮你填到脚本里。

---

## 四、第三步：上传到 GitHub（10分钟）

### 4.1 创建 GitHub 账号

打开 https://github.com 注册（免费）

### 4.2 创建仓库

1. 点击右上角 **+** → **New repository**
2. 填写：
   - **Repository name**: `touzi-kb`
   - **Private**: 可选
3. 点击 **Create repository**

### 4.3 上传代码

1. 在仓库页面点击 **uploading an existing file**
2. 打开 Finder，进入 `~/Desktop/投资大佬agent知识库/`
3. **选中 `website` 文件夹**，拖进去
4. 点击 **Commit changes**

---

## 五、第四步：连接 Vercel（10分钟）

### 5.1 注册 Vercel

1. 打开 https://vercel.com
2. 点击 **Sign Up**
3. 选择 **Continue with GitHub**

### 5.2 导入项目

1. 点击 **Add New** → **Project**
2. 找到你的仓库（`touzi-kb`）
3. 点击 **Import**
4. 配置：
   - **Framework Preset**: Other
   - **Root Directory**: `./`
   - **Build Command**: 留空
   - **Output Directory**: `.`
5. 点击 **Deploy**

### 5.3 获取地址

部署完成后，你会得到一个 `.vercel.app` 地址，这就是你的网站！

---

## 六、第五步：设置自动更新（5分钟）

### 6.1 在 GitHub 仓库添加 Cookie

1. 进入仓库 → **Settings** → **Secrets and variables** → **Actions**
2. 点击 **New repository secret**
3. Name 填：`XUEQIU_COOKIE`
4. Secret 填：你的 Cookie 值
5. 点击 **Add secret**

### 6.2 触发第一次爬取

1. 进入仓库 → **Actions** 标签
2. 点击左侧 **雪球爬取与网站更新**
3. 点击 **Run workflow**
4. 选择 **full**（首次全量）
5. 点击 **Run workflow**

---

## 七、后续使用

### 7.1 每月更新 Cookie

雪球 Cookie 每月过期：

1. 重新登录雪球，获取新 Cookie
2. GitHub 仓库 → **Settings** → **Secrets and variables** → **Actions**
3. 点击 `XUEQIU_COOKIE` 右边的编辑
4. 粘贴新 Cookie → 保存

### 7.2 手动触发更新

GitHub 仓库 → **Actions** → **雪球爬取与网站更新** → **Run workflow**

### 7.3 查看运行日志

点击 Actions 中的运行记录，可以看到：
- 爬取了多少条帖子
- 发现了哪些新公司
- 部署状态

## 八、自动化功能说明

### 8.1 自动发现新公司

爬虫会自动：
- 检测帖子中新提到的公司
- 记录首次提及时间
- 生成"新提及公司"标签

### 8.2 自动更新网站数据

每次爬取后自动：
- 更新公司列表
- 生成语录数据
- 重新部署网站

### 8.3 数据存储

```
raw/xueqiu/
├── duanyongping_posts.json      # 全部帖子
├── companies_config.json         # 公司配置
├── crawl_state.json             # 爬取状态
└── incremental_*.json           # 增量记录
```
