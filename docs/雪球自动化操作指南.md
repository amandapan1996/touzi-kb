# 雪球自动化 - 新手操作指南

> 本文档为零基础用户编写，按步骤操作即可。

---

## 一分钟概览

这个功能会自动：
1. 每周一凌晨自动爬取段永平的新帖子
2. 提取提到的公司、话题
3. 更新到网站上
4. 部署网站

**你需要做的**：
- 初始配置（5分钟，一次性）
- 每月更新Cookie（3分钟）

---

## 第一步：获取雪球Cookie（5分钟）

### 1.1 打开浏览器

用Chrome或Safari打开 https://xueqiu.com

### 1.2 登录账号

用你的雪球账号登录

### 1.3 打开开发者工具

- **Chrome**: 按 `Command + Option + I`（Mac）或 `F12`（Windows）
- **Safari**: 先菜单栏 → 偏好设置 → 高级 → 勾选"在菜单栏中显示开发菜单"，然后按 `Command + Option + I`

### 1.4 找到Cookie

1. 点击开发者工具顶部的 **Application** 标签
2. 左侧展开 **Cookies** → 点击 `https://xueqiu.com`
3. 找到名为 `xq_a_token` 的行
4. 复制 **Value** 列的值（一长串字母数字）

### 1.5 保存Cookie

把复制的Cookie值发给Claude Code，让我帮你填到脚本里。

---

## 第二步：上传到GitHub（3分钟）

### 2.1 创建GitHub账号

打开 https://github.com 注册账号（免费）

### 2.2 创建仓库

1. 点击右上角 **+** → **New repository**
2. 仓库名称填写：`touzi-agent-kb`
3. 选择 **Private**（私有）
4. 点击 **Create repository**

### 2.3 上传代码

在创建好的仓库页面：
1. 点击 **uploading an existing file**
2. 把 `投资大佬agent知识库` 文件夹里的**所有内容**拖进去
3. 点击 **Commit changes**

### 2.4 设置Cookie

1. 进入仓库 → **Settings** → **Secrets and variables** → **Actions**
2. 点击 **New repository secret**
3. Name填：`XUEQIU_COOKIE`
4. Secret填：你之前复制的Cookie值
5. 点击 **Add secret**

---

## 第三步：设置自动部署（2分钟）

### 3.1 连接Vercel

1. 打开 https://vercel.com
2. 用GitHub账号登录
3. 点击 **Add New** → **Project**
4. 找到你的 `touzi-agent-kb` 仓库，点击 **Import**
5. **Framework Preset** 选择 **Other**
6. **Root Directory** 保持默认
7. 点击 **Deploy**

### 3.2 获取Vercel配置

部署完成后：
1. 进入 Vercel 项目 → **Settings** → **General**
2. 复制 **Vercel Project ID**
3. 复制 **Vercel Team ID**（在 Account Settings 里）

### 3.3 生成Vercel Token

1. 打开 https://vercel.com/account/tokens
2. 点击 **Create Token**
3. Name随便填，如 `crawl-bot`
4. Scope选择 **Full Account**
5. 点击 **Create**
6. 复制生成的Token

### 3.4 添加到GitHub Secrets

回到GitHub仓库 → **Settings** → **Secrets and variables** → **Actions**，添加：

| Name | Value |
|------|-------|
| VERCEL_TOKEN | 你生成的Token |
| VERCEL_ORG_ID | 你的Team ID |
| VERCEL_PROJECT_ID | 你的Project ID |

---

## 第四步：测试运行（1分钟）

### 4.1 手动触发

1. 进入GitHub仓库 → **Actions** 标签
2. 点击左侧 **雪球爬取与网站更新**
3. 点击 **Run workflow**
4. 选择 **incremental** 或 **full**
5. 点击 **Run workflow**

### 4.2 查看结果

- 点击运行的workflow查看日志
- 绿色的勾表示成功
- 红色的X表示失败，可以查看错误信息

---

## 每月任务：更新Cookie

雪球Cookie每月会过期，需要更新：

1. 重新登录雪球，按上面的步骤获取新的Cookie
2. 进入GitHub仓库 → **Settings** → **Secrets and variables** → **Actions**
3. 点击 `XUEQIU_COOKIE` 右边的编辑按钮
4. 粘贴新Cookie，点击保存

---

## 常见问题

### Q: 提示 "Cookie已过期"
A: Cookie过期了，按上面的步骤重新获取并更新。

### Q: 提示 "请求失败 403"
A: 可能是网络问题或Cookie无效，重试一次或更新Cookie。

### Q: 网站没更新
A: 检查Actions里的运行日志，看是否有错误。

### Q: 想手动触发更新
A: GitHub仓库 → Actions → 雪球爬取与网站更新 → Run workflow
