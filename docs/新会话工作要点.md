## 第一步：配置 Cookie

### 1.1 获取 Cookie

1. 用 Chrome 打开 https://xueqiu.com 并登录
2. 按 `Command + Option + I` 打开开发者工具
3. 点击顶部 **Application** 标签
4. 左侧点击 **Cookies** → `https://xueqiu.com`
5. 找到 `xq_a_token`，复制 **Value** 列的值

### 1.2 填入脚本

编辑 `scripts/crawl_xueqiu.py`，找到第 27 行：

```python
COOKIE = ""  # TODO: 填入你的Cookie
```

把 Cookie 粘贴进去。

---

## 第二步：部署到 Vercel

### 2.1 创建 GitHub 仓库

1. 打开 https://github.com，注册/登录
2. 点击 **+** → **New repository**
3. 仓库名：`touzi-kb`
4. 点击 **Create repository**

### 2.2 上传网站代码

1. 进入仓库，点击 **uploading an existing file**
2. 打开 Finder，进入 `~/Desktop/投资大佬agent知识库/`
3. **选中 `website` 文件夹**，拖进去
4. 点击 **Commit changes**

### 2.3 连接 Vercel

1. 打开 https://vercel.com
2. 用 GitHub 账号登录
3. 点击 **Add New** → **Project**
4. 找到 `touzi-kb` 仓库，**Import**
5. 配置：
   - **Framework Preset**: Other
   - **Root Directory**: `./`
   - **Build Command**: 留空
   - **Output Directory**: `.`
6. 点击 **Deploy**

### 2.4 添加 Cookie 到 GitHub

1. 进入 GitHub 仓库 → **Settings** → **Secrets and variables** → **Actions**
2. 点击 **New repository secret**
3. Name: `XUEQIU_COOKIE`
4. Secret: 粘贴 Cookie 值
5. 点击 **Add secret**


## 常见问题

### Q: Cookie 过期了怎么办？
A: 每月重新获取一次，粘贴到 GitHub Secrets

### Q: 网站没有自动更新？
A: GitHub 仓库 → Actions → 查看运行日志

### Q: 想手动触发更新？
A: GitHub 仓库 → Actions → 雪球爬取与网站更新 → Run workflow
