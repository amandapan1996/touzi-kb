# 雪球帖子接入指南

> 本文档说明如何获取段永平的雪球帖子数据。

---

## 方法一：使用脚本爬取（推荐）

### 步骤 1：获取Cookie

1. 在浏览器中登录雪球 (https://xueqiu.com)
2. 按 F12 打开开发者工具
3. 切换到 Application 标签
4. 左侧选择 Cookies -> https://xueqiu.com
5. 找到 `xq_a_token` 或 `u` 相关的cookie
6. 复制完整的cookie字符串

### 步骤 2：配置脚本

编辑 `scripts/crawl_xueqiu.py`，将Cookie填入：

```python
COOKIE = "你的Cookie值"
```

### 步骤 3：运行脚本

```bash
cd ~/Desktop/投资大佬agent知识库
source .venv/bin/activate
python3 scripts/crawl_xueqiu.py
```

### 步骤 4：消化数据

获取的数据会保存到 `raw/xueqiu/duanyongping_posts_YYYYMMDD.json`

---

## 方法二：手动导出

如果你有雪球账号权限：

1. 登录段永平的雪球主页
2. 使用浏览器插件导出页面
3. 将导出的HTML/JSON文件放到 `raw/xueqiu/` 目录

---

## 方法三：API直连

雪球提供了一些公开API，可以尝试：

```
# 获取用户信息
GET https://xueqiu.com/v4/statuses/user_timeline.json?user_id=5497652514

# 需要Cookie认证
```

---

## Cookie格式示例

```
xq_a_token=xxxx; xq_r_token=xxxx; u=xxxx; ...
```

完整的Cookie字符串，多个key-value用分号分隔。

---

## 注意事项

1. **Cookie有效期**：雪球Cookie通常有7-30天有效期，过期需重新获取
2. **请求频率**：脚本已有1秒间隔，避免被封
3. **法律合规**：请遵守雪球的使用条款，仅用于个人学习
