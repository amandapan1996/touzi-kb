# 段永平投资案例分析

> 本页分析段永平的经典投资案例，提炼可学习的投资逻辑。

---

## 案例一：网易 (2001-2002)

### 基本信息

| 项目 | 内容 |
|------|------|
| 投资时间 | 2001-2002年 |
| 买入价格 | < 1美元 |
| 持仓周期 | 约5年 |
| 最终收益 | 超过100倍 |
| 代表性持仓 | 重仓 |

---

### 投资背景

**宏观环境**：
- 2000年互联网泡沫破裂
- 纳斯达克指数从5048点跌到1100点
- 互联网公司普遍暴跌90%以上

**公司情况**：
- 网易从上市初的15美元跌到0.8美元
- 被SEC调查，一度面临摘牌风险
- 主营业务游戏还未盈利

---

### 投资分析

#### 为什么投网易？

**1. 商业模式理解**
```
✓ 段永平做过步步高，理解游戏行业
✓ 游戏是个好生意：边际成本低、可复制
✓ 中国游戏市场潜力巨大
✓ 短信业务也有增长
```

**2. 财务分析**
```
现金: 约2亿美元 (超过市值)
负债: 几乎没有
营收: 持续增长
估值: 现金超过市值 = 严重低估
```

**3. 风险评估**
```
SEC问题 → 可控，不会致命
摘牌风险 → 即使摘牌，公司本身还在
游戏前景 → 看好但不确定
→ 风险收益比极佳
```

#### 投资决策

> "我当时觉得赔光的概率很小，赚大钱的概率很大。"

**风险收益分析**：
| 情景 | 概率 | 结果 |
|------|------|------|
| 网易倒闭 | 10% | 亏光 |
| 勉强存活 | 30% | 保本或小赚 |
| 正常发展 | 40% | 几倍 |
| 成功发展 | 20% | 几十倍 |

---

### 案例启示

#### 可量化的规则

```yaml
案例一规则:
  触发条件:
    - 市值 < 现金
    - 主营业务有前景
    - 管理层可信
  仓位:
    - 可以重仓
    - 极端低估时加大仓位
  止损:
    - 基本面没变就不卖
```

#### 思维框架

```
1. 极端悲观时买入
2. 看懂商业模式再买
3. 计算下行风险和上行空间
4. 风险可控时可以重仓
5. 耐心等待价值回归
```

## 案例二：苹果 (2011年至今)

### 基本信息

| 项目 | 内容 |
|------|------|
| 首次买入 | 2011年 |
| 买入依据 | 看好智能手机 + 商业模式 |
| 持仓周期 | 持续持有 |
| 买入逻辑 | 生态护城河 + 管理层 |
| 当前状态 | 重仓持有 |

---

### 投资分析

#### 为什么投苹果？

**1. 能力圈内**
```
✓ 段永平长期关注科技行业
✓ 理解消费电子的商业模式
✓ 理解品牌溢价
✓ 理解生态系统的价值
```

**2. 商业模式优秀**
```
收入结构:
  - iPhone: 约50%
  - 服务: 约20% (高利润率)
  - 其他硬件: 约30%

护城河:
  - iOS生态系统 (极高转换成本)
  - 品牌溢价
  - 定价权
  - 生态闭环
```

**3. 管理层**
```
库克接任后:
  ✓ 供应链管理卓越
  ✓ 股东回报意识强
  ✓ 大量回购股票
  ✓ 诚信可靠
```

**4. 估值合理**
```
2011年PE: 约10-15倍
当时市场给的估值偏低
段永平认为被低估
```

---

### 持有逻辑

> "持有苹果的逻辑一直没变：生态护城河还在，管理层还在做对的事。"

**持有检查点**：
```
[ ] iOS生态被安卓取代？ → 否
[ ] 创新能力消失？ → 否
[ ] 管理层不本分？ → 否
[ ] 价格太离谱？ → 评估中
```

---

### 案例启示

#### 可量化的规则

```yaml
案例二规则:
  适用场景:
    - 顶级商业模式
    - 强大护城河
    - 可持续竞争优势
  买入信号:
    - PE < 历史中位数
    - 负面消息导致下跌
    - 市场整体恐慌
  持有原则:
    - 逻辑没变就一直持有
    - 定期检查护城河是否还在
    - 不因短期波动卖出
```

## 案例三：茅台 (长期持有)

### 投资逻辑

**1. 品牌护城河**
```
✓ 中国第一高端白酒品牌
✓ 无法复制的赤水河水和气候
✓ 历史积淀的品牌价值
✓ 越陈越香的产品特性
```

**2. 商业模式**
```
高毛利: > 90%
高净利: > 50%
先款后货: 无应收账款
存货升值: 不怕积压
```

**3. 定价权**
```
飞天茅台建议零售价: 1499元
实际成交价: 2000-3000元
溢价率: > 100%
```

---

### 量化指标

| 指标 | 茅台 | 行业平均 |
|------|------|----------|
| 毛利率 | 90%+ | 60% |
| 净利率 | 50%+ | 15% |
| ROE | 30%+ | 10% |
| 现金流 | 优秀 | 一般 |
| 预收款 | 高 | 低 |

```

**3. 马化腾**

✓ 持续回购
```

## 案例启示总结

### 通用投资框架

```
段永平投资框架:

1. 选股标准
   ├── 懂 → 能理解商业模式
   ├── 好 → 商业模式优秀
   ├── 本 → 管理层本分
   └── 对 → 价格合理

2. 买入原则
   ├── 极端低估时重仓
   ├── 合理价格时轻仓
   └── 不追高

3. 持有原则
   ├── 逻辑不变就不卖
   ├── 越跌越买（如果有现金）
   └── 忽略短期波动

4. 卖出原则
   ├── 基本面变坏
   ├── 估值离谱
   └── 发现更好的机会
```

### 可复制的模式

1. **极端悲观时买入**
   - 市场恐慌 → 机会
   - 个股危机 → 具体分析
   - 计算风险收益比

2. **能力圈内深耕**
   - 专注于懂的行业
   - 深入研究少数公司
   - 建立自己的能力圈

3. **与靠谱的人同行**
   - 管理层是重要的考量
   - 本分 > 能力
   - 宁可不投，不投错
