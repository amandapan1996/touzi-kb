# 能力圈 (Circle of Competence)

> "投资最重要的事是知道自己知道什么，更重要的是知道自己不知道什么。" —— 段永平

---

## 核心理念

能力圈原则是价值投资的核心之一。段永平强调：

> "不懂的东西不要碰。看不懂的公司就放在一边，等看懂了再说。"

### 能力圈的本质

```
能力圈 = 真正理解的事物范围

- 不是你碰过的范围
- 不是你听说过的范围
- 是你真正能理解本质的范围
```

---

## 能力圈的两个维度

### 1. 知道自己懂什么

**判断标准**：
| 懂 | 不懂 |
|------|------|
| 能说清楚商业模式 | 只会说"是个好公司" |
| 能估算大致价值区间 | 不知道值多少钱 |
| 能识别主要风险 | 不知道风险在哪 |
| 能理解竞争格局 | 分不清竞争对手 |

### 2. 知道自己不懂什么

**同样重要**：
> "知道自己不知道，比知道自己知道更重要。"

**不懂的表现**：
- 无法回答"这家公司靠什么赚钱"
- 无法解释"为什么是这家公司而不是竞争对手"
- 无法判断"值多少钱"

## 段永平的能力圈实践

### 只投资熟悉的领域

> "我只投 TMT（科技、媒体、通信）和消费类公司，因为我在这些领域干了20多年。"

**段永平的能力圈**：
```
✓ 消费电子（OPPO、vivo、小天才）
✓ 互联网社交（腾讯）
✓ 游戏（网易）
✓ 白酒（茅台）
✓ 消费品品牌（苹果）
✓ 商业零售（拼多多）

✗ 金融（不太懂，不投）
✗ 制造业（没有优势）
✗ 医药（看不懂）
```

### 网易投资案例

**为什么投网易？**

1. **懂游戏**：段永平做过步步高，知道游戏行业
2. **懂互联网**：理解互联网的盈利模式
3. **看懂了机会**：2001年互联网泡沫，股价大跌
4. **算清楚了价值**：现金超过市值

> "我当时觉得网易的股价被严重低估了，而且我懂它的业务。"

## 如何扩大能力圈

### 1. 持续学习

> "能力圈可以慢慢扩大，但要以'真懂'为前提。"

**扩大方法**：
- 大量阅读行业报告
- 关注行业发展趋势
- 亲身体验产品和服务
- 与行业专家交流

### 2. 谨慎扩大

**段永平的警告**：
> "能力圈扩大的代价是可能变得不精。不要为了扩大而扩大。"

**扩大原则**：
```
第一步：了解 → 知道行业的基本情况
第二步：理解 → 理解行业的本质逻辑
第三步：精通 → 能做出投资判断

只做到第一步就想投资？那叫投机。
```

### 3. 诚实地自我评估

**评估问题清单**：
- [ ] 我能向一个10岁小孩解释这家公司吗？
- [ ] 我能预测这家公司5年后的样子吗？
- [ ] 我知道这家公司可能失败的原因吗？
- [ ] 如果股市关闭5年，我还愿意持有吗？

## 能力圈与风险控制

### 不懂的风险

| 风险 | 后果 |
|------|------|
| 买了不懂的公司 | 可能血本无归 |
| 无法判断估值 | 可能买贵了 |
| 无法识别风险 | 可能遇到黑天鹅 |
| 无法跟踪 | 可能错过变坏信号 |

### 能力圈是最好的风控

> "不懂的东西不碰，就是最好的风控。"

**风险管理层次**：
```
第一层：能力圈内投资（最重要）
第二层：适度分散
第三层：仓位控制
第四层：止损（最后手段）
```

## 能力圈 vs 分散投资

### 矛盾吗？

**表面矛盾**：
- 能力圈 → 集中投资自己懂的
- 分散投资 → 不把鸡蛋放一个篮子

**实际统一**：
> "能力圈内的集中，本身就是分散。"
> "你在自己的能力圈里有5-10家公司可选，那就已经足够分散了。"

### 段永平的做法

> "我一般持有 3-5 家公司，都是我很懂的。"

**他的逻辑**：
- 3-5 家已经很懂的公司
- 每家都有深入研究
- 集中但不过度
- 不追求覆盖所有行业

## 能力圈与机会成本

### 错过机会不是损失

> "错过机会不叫损失，因为那个机会本来就不是你的。"
> "只有做错才是真正的损失。"

**对比**：
| 错过能力圈内机会 | 错过能力圈外机会 |
|------------------|------------------|
| 应该反思 | 不需要后悔 |
| 说明研究不够 | 本来就不该碰 |
| 是损失 | 不是损失 |

### 等待的艺术

> "好机会是等出来的。不要因为焦虑就乱投资。"

**等待的底气**：
- 知道自己懂什么
- 知道自己不懂什么
- 有耐心等待好价格
- 不被 FOMO（Fear Of Missing Out）驱动
