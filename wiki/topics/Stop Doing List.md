# Stop Doing List (不为清单)

> "做对的事情，把事情做对。如果事情本身是错的，做得越好越糟糕。" —— 段永平

---

## 核心理念

**Stop Doing List** 是段永平最重要的投资哲学之一。

与 To-Do List（要做的事）相比，更重要的是知道**不应该做什么**。

### 为什么要做"不为清单"？

1. **避免致命错误**
   - 投资中亏损50%需要100%才能回本
   - 一次重大失误可能毁掉整个投资组合

2. **保持专注**
   - 人的精力有限
   - 减少干扰，专注于核心机会

3. **长期复利**
   - 避免亏损 + 持续盈利 = 复利效应
   - 活得久比赚得快重要

## 段永平的"不为清单"

### 投资方面

| 不为 | 原因 |
|------|------|
| **不做空** | 风险收益不对称，潜在亏损无限 |
| **不借钱买股票** | 会影响判断，造成致命错误 |
| **不做不懂的东西** | 不懂的东西无法判断风险 |
| **不跟风** | 别人恐慌时我也不一定买 |
| **不迷信权威** | 包括巴菲特说的话（要自己理解） |
| **不做短线** | 无法预测短期股价 |
| **不纠结于宏观** | 宏观无法预测，也不重要 |
| **不预测市场** | 市场短期是随机的 |

### 商业方面

| 不为 | 解释 |
|------|------|
| **不欺骗消费者** | 短期可能占便宜，长期必受惩罚 |
| **不偷工减料** | 质量是底线 |
| **不恶性竞争** | 打价格战对谁都没好处 |
| **不上市圈钱** | 不是不上市，是不为圈钱而上市 |
| **不盲目扩张** | 能力圈之外的机会不是机会 |

## 段永平的决策原则

### 决策框架

```
1. 这是对的事情吗？     ← 先判断方向
2. 这是我懂的东西吗？   ← 再确认能力圈
3. 我能承受这个风险吗？  ← 最后评估风险
```

### 宁可错过，不能做错

> "错过没什么可怕的，可怕的是做错了。"

| 错过 | 做错 |
|------|------|
| 机会成本 | 本金损失 |
| 可以再找机会 | 可能血本无归 |
| 心理可接受 | 影响生活、家庭 |

## Stop Doing List 的延伸

### 巴菲特的类似观点

**能力圈原则**：
> "知道你不知道什么比知道你知道什么更重要。"

**二十打孔规则**：
> "一生只需要做二十个重大投资决策，机会自然会来。"

### 芒格的"反过来想"

> "我只想知道我会死在哪里，然后永远不去那里。"

## 如何建立自己的 Stop Doing List

### Step 1: 回顾错误
- 列出过去犯过的投资错误
- 分析每个错误的原因
- 提炼出不应该做的事

### Step 2: 学习他人
- 阅读段永平、巴菲特、芒格的观点
- 理解背后的逻辑
- 转化为自己的原则

### Step 3: 定期更新
- 每季度审视清单
- 根据新经验调整
- 保持简洁，重点突出

### Step 4: 严格执行
- 把清单放在显眼位置
- 每次投资决策前检查
- 养成习惯
