# 平常心 (Common Sense / Zen Mind)

> "平常心就是理性，就是不做傻事。" —— 段永平

---

## 什么是平常心？

平常心是段永平投资哲学的基石之一。它不是"无所谓"的态度，而是一种**理性的、长期导向的思维方式**。

### 平常心不是什么

| 不是 | 解释 |
|------|------|
| **无所谓** | 对结果漠不关心 |
| **躺平** | 不努力，不追求 |
| **听天由命** | 不做分析就做决策 |
| **赌博心态** | 追求一夜暴富 |

### 平常心是什么

| 是 | 解释 |
|------|------|
| **理性** | 不被情绪左右 |
| **客观** | 实事求是看问题 |
| **耐心** | 愿意等待好的机会 |
| **淡定** | 不为短期波动所动 |
| **专注** | 专注于自己能控制的事 |

---

## 平常心的具体表现

### 1. 不追求暴富

> "我没想过要赚快钱，慢慢变富挺好的。"

**心态对比**：
| 追求暴富 | 平常心 |
|----------|--------|
| 杠杆、借钱 | 闲钱投资 |
| 追涨杀跌 | 逆向投资 |
| 频繁交易 | 长期持有 |
| 焦虑不安 | 睡得着觉 |

### 2. 不为短期波动所动

> "股价跌了我不会卖，涨了我也不会急着卖。"

**面对波动的态度**：
- 跌了：检查基本面有没有变，没变就不管
- 涨了：评估是否高估，高估就卖，没高估就持有
- 不跌不涨：继续持有，不浪费时间

### 3. 承认不懂，不硬撑

> "这个我不懂，我不会因为别人说好就买。"

**不懂就问/不买**：
- 承认不懂是智慧
- 不懂装懂是愚蠢
- 等待懂了再行动

### 4. 不跟别人比较

> "我从不跟别人比收益率，那样没意义。"

**比较的问题**：
- 每个人的情况不同
- 每个人的能力圈不同
- 每个人的风险承受能力不同

## 如何培养平常心

### 1. 用闲钱投资

> "我用三年不用的钱投资，所以不着急。"

**财务准备**：
```
短期要用的钱 → 银行存款、货币基金
中期要用的钱 → 债券、理财
长期不用的钱 → 股票投资
```

### 2. 深入理解投资对象

> "看懂一家公司后，股价波动就不会影响你了。"

**理解带来淡定**：
- 懂 → 知道值多少钱 → 不怕跌
- 懂 → 知道能涨多少 → 不急于卖

### 3. 降低预期

> "我对年化收益没有太高期望，能跑赢通胀就行。"

**合理预期**：
- 巴菲特长期年化约 20%
- 普通人 10-15% 已是不错
- 追求 30%、50% 年化往往亏损

### 4. 分散但不过度

> "我一般持有 3-5 家公司，太多了顾不过来。"

**适度分散**：
- 太少 → 风险集中
- 太多 → 无力研究
- 适度 → 3-10 家

## 10-10-10 法则

段永平常用的决策框架：

| 时间维度 | 问题 | 作用 |
|----------|------|------|
| **10分钟后** | 这个决定还重要吗？ | 避免小事影响情绪 |
| **10个月后** | 我会怎么看这个决定？ | 中期视角 |
| **10年后** | 我会后悔吗？ | 长期导向 |

### 应用场景

**买入决策**：
- 10分钟后：现在买入会安心吗？
- 10个月后：如果跌了20%，我会后悔吗？
- 10年后：这个公司还会存在吗？

**卖出决策**：
- 10分钟后：卖出后我会后悔吗？
- 10个月后：这个逻辑变了吗？
- 10年后：现在卖是明智的吗？

## 平常心与投资回报

| 没有平常心 | 保持平常心 |
|------------|------------|
| 频繁交易，贡献手续费 | 长期持有，减少摩擦 |
| 追涨杀跌，高买低卖 | 逆向投资，低买高卖 |
| 焦虑影响生活 | 投资让生活更好 |
| 容易做出错误决策 | 决策质量更高 |
