## 一、选股规则

### 1.1 商业模式筛选

```yaml
规则编号: DYP-001
规则名称: 优秀商业模式筛选
规则类型: 选股过滤器
可量化: 是

筛选条件:
  - 一句话说清楚商业模式
  - 毛利率 >= 30%
  - 净利率 >= 10%
  - ROE >= 15%
  - 现金流健康 (经营现金流 > 净利润)

排除条件:
  - 靠政策垄断吃饭的
  - 需要不断烧钱维持的
  - 客户没有忠诚度的
  - 靠关联交易赚钱的
```

### 1.2 护城河筛选

```yaml
规则编号: DYP-002
规则名称: 护城河验证
规则类型: 选股过滤器
可量化: 部分可量化

护城河类型:
  - 无形资产: 品牌、专利、特许经营权
  - 转换成本: 用户换竞品代价高
  - 网络效应: 用户越多越有价值
  - 成本优势: 规模效应、地理位置

可量化指标:
  - 品牌护城河: 市值/营收 (品牌溢价)
  - 转换成本: 客户留存率 >= 90%
  - 网络效应: 用户数增速 vs 收入增速
  - 成本优势: 毛利率持续高于行业平均
```

### 1.3 管理层筛选

```yaml
规则编号: DYP-003
规则名称: 管理层本分度评估
规则类型: 选股过滤器
可量化: 部分可量化

正面指标:
  - 历史上有"本分"的行为记录
  - 管理层持股比例合理
  - 愿意回购股票
  - 信息披露透明及时

负面指标:
  - 有财务造假前科
  - 关联交易频繁且不透明
  - 高管薪酬与业绩脱钩
  - 不诚信的过往记录

排除条件:
  - 管理层不诚信的
  - 有财务问题的
  - 不为股东考虑的
```

## 二、买入规则

### 2.1 安全边际规则

```yaml
规则编号: DYP-101
规则名称: 安全边际买入
规则类型: 买入条件
可量化: 是

计算方法:
  内在价值估算: DCF或简化估值
  安全边际: (内在价值 - 当前价格) / 内在价值

买入条件:
  - 安全边际 >= 30% → 强烈买入
  - 安全边际 >= 20% → 买入
  - 安全边际 >= 10% → 可考虑
  - 安全边际 < 10% → 观望
```

### 2.2 估值区间规则

```yaml
规则编号: DYP-102
规则名称: PE估值区间
规则类型: 买入条件
可量化: 是

参考指标:
  - 历史PE区间
  - 行业平均PE
  - 成长性调整

段永平偏好:
  - 不追求精确估值
  - 模糊的正确即可
  - 给定合理区间
  - 在低估区间分批买入
```

## 三、卖出规则

### 3.1 卖出条件

```yaml
规则编号: DYP-201
规则名称: 卖出决策规则
规则类型: 卖出条件
可量化: 是

卖出条件 (满足任一):
  - 基本面恶化:
    - 商业模式被破坏
    - 护城河消失
    - 管理层不本分
  - 估值过高:
    - PE远超历史区间
    - 市值远超内在价值
  - 发现更好的机会:
    - 能力圈内更好的标的
    - 风险收益比更优

卖出规则:
  - 不因股价短期波动卖出
  - 不因别人推荐卖出
  - 不因恐慌卖出
  - 只因逻辑变化卖出
```

### 3.2 不卖出规则

```yaml
规则编号: DYP-202
规则名称: 坚定持有规则
规则类型: 卖出限制
可量化: 否

以下情况不卖出:
  - 股价下跌但基本面没变
  - 短期业绩不佳但长期逻辑在
  - 市场整体下跌
  - 听到各种负面消息但未经证实
  - 别人说不好但自己没想明白

"只要逻辑没变，越跌越买"
```

## 四、仓位管理规则

### 4.1 集中度规则

```yaml
规则编号: DYP-301
规则名称: 仓位集中规则
规则类型: 仓位管理
可量化: 是

规则:
  - 单一个股仓位: <= 40%
  - 前三大持仓合计: <= 70%
  - 持仓数量: 3-10只
  - 闲钱投资，不用杠杆
```

### 4.2 买入节奏

```yaml
规则编号: DYP-302
规则名称: 分批买入规则
规则类型: 仓位管理
可量化: 是

规则:
  - 首次买入: 30%
  - 回调10%: 20%
  - 回调20%: 30%
  - 现金备用: 20%
```
