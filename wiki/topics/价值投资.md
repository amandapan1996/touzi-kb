# 价值投资 (Value Investing)

> "投资就是买企业，买企业就是买未来现金流折现。" —— 段永平

---

## 价值投资的核心定义

### 什么是价值投资？

**段永平的定义**：
> "价值投资就是用低于价值的价格买进，然后持有，等待价值回归或成长。"

**本质**：
```
价值投资 = 买入折扣 + 内在价值 + 成长价值

安全边际 = 价格与价值的差距
内在价值 = 企业未来现金流折现
成长价值 = 企业成长的部分
```

### 价值投资的四个核心

| 核心 | 内容 |
|------|------|
| **买股票就是买公司** | 把股票当作公司的一部分所有权 |
| **理解内在价值** | 估算公司值多少钱 |
| **安全边际** | 用折扣价买入 |
| **市场先生** | 利用市场的非理性 |

## 段永平的价值投资框架

### 1. 买股票就是买公司

> "很多人把股票当作筹码，我从来不这么看。"

**投资 vs 投机**：
| 投资 | 投机 |
|------|------|
| 买公司的一部分 | 买价格的波动 |
| 关心公司价值 | 关心股价走势 |
| 长期持有 | 短期交易 |
| 分析企业 | 分析图形 |

### 2. 理解商业模式

> "买股票之前，先问自己：这家公司的商业模式是什么？"

**好商业模式特征**：
- [ ] 一句话能说清楚怎么赚钱
- [ ] 靠产品或服务本身赚钱，不靠关联交易
- [ ] 赚钱不辛苦（高利润率）
- [ ] 有护城河保护利润
- [ ] 不需要大量资本投入维持

**差商业模式特征**：
- 靠政策保护或垄断
- 需要不断烧钱维持
- 毛利率低、周转慢
- 客户没有忠诚度

### 3. 内在价值估算

**内在价值的定义**：
> "内在价值是企业在剩余存续期内能够产生的现金流的折现值。"

**简化版思维**：
```
第一步：估算未来5-10年的自由现金流
第二步：给一个合理的折现率
第三步：得到一个大致的价值区间
第四步：留足安全边际
```

**段永平的方法**：
> "我不追求精确的数字，只求模糊的正确。"
> "如果我用 DCF 算出来是 100 亿，我不会纠结它是 95 亿还是 105 亿。"

### 4. 安全边际

**什么是安全边际？**
> "安全边际就是用5块钱买价值10块钱的东西。"

**为什么需要安全边际？**
- 估值是估计，不是精确计算
- 企业经营有不确定性
- 防止判断错误造成损失

**段永平的安全边际标准**：
> "一般要有 30-50% 的折扣才会出手。"

## 价值投资 vs 其他策略

### 对比表格

| 维度 | 价值投资 | 成长投资 | 趋势投资 |
|------|----------|----------|----------|
| **关注点** | 内在价值 | 未来增长 | 股价趋势 |
| **持有期** | 长期（5-10年+） | 中期（2-5年） | 短期（几周-几个月） |
| **估值方法** | PE、PB、DCF | PEG、PS | 技术分析 |
| **典型代表** | 巴菲特、段永平 | 费雪 | 索罗斯 |
| **风险特征** | 低波动 | 中波动 | 高波动 |

### 段永平的选择

> "我是价值投资者，不是成长投资者。"
> "但好的价值投资要包含成长性。"

**理解**：
- 价值投资不等于买低估的传统公司
- 成长性是价值的一部分
- 关键是好公司 + 好价格

## 价值投资的心理障碍

### 1. 锚定效应

**问题**：股价已经跌了50%，还能买吗？

**答案**：
> "不要看股价跌了多少，要看现在值不值。"
> "跌了不代表便宜，要看估值。"

### 2. 损失厌恶

**问题**：持有的股票跌了20%，要不要卖？

**答案**：
> "先问自己：当初买的逻辑变了吗？"
> "如果没变，跌了反而应该考虑加仓。"
> "如果变了，不管盈亏都应该卖。"

### 3. 从众心理

**问题**：大家都在卖，我要不要也卖？

**答案**：
> "别人恐慌的时候，你应该更理性。"
> "但'更理性'不是'逆向操作'，而是'客观分析'。"

## 段永平的价值投资案例

### 网易 (2001-2002)

**背景**：
- 互联网泡沫破裂
- 网易股价从30美元跌到1美元以下
- 被SEC调查，一度面临摘牌

**段永平的分析**：
1. **商业模式**：游戏+短信，当时很看好
2. **财务状况**：现金充足，超过市值
3. **估值**：严重低估
4. **风险**：SEC 问题可控

**结果**：持有多年，收益超过100倍

---

### 苹果 (2011年至今)

**买入逻辑**：
1. **懂**：一直在关注科技行业
2. **商业模式**：生态系统的护城河
3. **管理层**：库克接任后更加稳健
4. **估值**：当时 PE 不高

**持有逻辑**：
- 持续创新
- 强大的现金流
- 管理层值得信任
- 商业模式没有改变

## 价值投资者的特质

### 需要的性格

| 特质 | 为什么重要 |
|------|------------|
| **耐心** | 好价格需要等待 |
| **独立思考** | 不被市场左右 |
| **理性** | 不被情绪控制 |
| **逆向思维** | 人弃我取 |
| **专注** | 深入研究少数公司 |

### 需要避免的性格

| 特质 | 危害 |
|------|------|
| **急躁** | 想赚快钱，容易亏损 |
| **从众** | 追涨杀跌 |
| **过度自信** | 高估自己的能力 |
| **贪婪** | 持有到泡沫 |
| **恐惧** | 底部割肉 |
